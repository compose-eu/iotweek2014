The Code

All our code is up on GitHub for your perusal. We have examples of the Arduino sketches for running on the Spark.io board and Picus as well as the html for this page and the heatmap generation.

The Platforms

We used the COMPOSE platform in the cloud. We streamed the data from the sensors into Compose which stored them in an object oriented way then the website can display the values by requesting the data back from Compose in an API fashion. For the sensors we used the Spark.io platform and OpenPicus devlopment boards.

The Future

While we have only a handful of sensors in play right now, with more sensors we could deploy them real fast and build up larger maps with finer grained resolution. By publishing this information outward we feel we could help in new and exciting ways. For example, emergency services could have a remote view of the state of the building and this could help a central dispatch station to direct personnel in the most accurate and safe pathways through the building.
